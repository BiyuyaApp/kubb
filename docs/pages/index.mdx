<h1 className="text-center font-extrabold md:text-5xl mt-8">Kubb</h1>
<h2 className="text-center font-extrabold md:text-2xl mt-8">Codegen plugins</h2>

<Callout>
 With Kubb you can create and use plugins that will generate code based on a specific input. 
</Callout>


Instead of using multiple libraries to create your code generated files you can use Kubb with all of it's plugins to generate your code.
All of Kubb's logic is based on a core library(reading files, writing files, the plugin system, ...), which means you can use Kubb to create your own code generated logic.
