# Installation

To get started with generating code you need to install Kubb.

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash copy
    pnpm i -D @kubb/cli @kubb/core
    ```
  </Tab>
    <Tab>
    ```bash copy
    npm i -D @kubb/cli @kubb/core
    ```
  </Tab>
    <Tab>
    ```bash copy
    yarn add -D @kubb/cli @kubb/core
    ```
  </Tab>
</Tabs>

Then, you need to create a [configuration](/docs/getting-started/configuration) file.
