# Installation

You can install Kubb via [NPM](https://www.npmjs.com/).

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash filename="cli" copy
    pnpm i -D @kubb/cli @kubb/core
    ```
  </Tab>
    <Tab>
    ```bash filename="cli" copy
    npm i -D @kubb/cli @kubb/core
    ```
  </Tab>
    <Tab>
    ```bash filename="cli" copy
    yarn add -D @kubb/cli @kubb/core
    ```
  </Tab>
</Tabs>


Kubb also publishes a canary version on every commit. To install the canary:

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash filename="cli" copy
    pnpm i -D @kubb/cli@canary @kubb/core@canary
    ```
  </Tab>
    <Tab>
    ```bash filename="cli" copy
    npm i -D @kubb/cli@canary @kubb/core@canary
    ```
  </Tab>
    <Tab>
    ```bash filename="cli" copy
    yarn add -D @kubb/cli@canary @kubb/core@canary
    ```
  </Tab>
</Tabs>



The next step will be to create a [configuration](/docs/getting-started/configuration) file.
