# @kubb/swagger-react-query

With the Swagger React-Query plugin you can create queryHooks for the definitions inside a Swagger file.

## Installation

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash copy
    pnpm i -D @kubb/swagger-react-query
    ```
  </Tab>
    <Tab>
    ```bash copy
    npm i -D @kubb/swagger-react-query
    ```
  </Tab>
    <Tab>
    ```bash copy
    yarn add -D @kubb/swagger-react-query
    ```
  </Tab>
</Tabs>

## Options


### output
Output to save the ReactQuery hooks.

Type: `string` <br/>
Default: `"hooks"`
