# @kubb/swagger-zodios

With the Swagger zodios plugin you can use [zodios](https://github.com/ecyrbe/zodios) to validate your schema's based on a Swagger file.

## Installation

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash copy
    pnpm i -D @kubb/swagger-zodios
    ```
  </Tab>
    <Tab>
    ```bash copy
    npm i -D @kubb/swagger-zodios
    ```
  </Tab>
    <Tab>
    ```bash copy
    yarn add -D @kubb/swagger-zodios
    ```
  </Tab>
</Tabs>

## Options


### output
Output to save the zodios instance.

Type: `string` <br/>
Default: `"zodios.ts"`
