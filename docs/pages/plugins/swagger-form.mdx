# @kubb/swagger-form

With the Swagger Form plugin you can create: 
- [react-hook-form](https://react-hook-form.com) hooks based on an operation in the Swagger file.

## Options


### output
Output to save the react-hook-form hooks.

Type: `string` <br/>
Default: `"hooks"`

### groupBy
Group the react-hook-form hooks based on the provided name.

#### type
Tag will group based on the operation tag inside the Swagger file.

Type: `'tag'` <br/>
Required: `true`

#### output
Relative path to save the grouped react-hook-form hooks.
`{{tag}}` will be replaced by the current tagName.

Type: `string` <br/>
Example: `hooks/{{tag}}Controller` => `hooks/PetController` <br/>
Default: `${output}/{{tag}}Controller`

#### exportAs
Name to be used for the `export * as {{exportAs}} from './`

Type: `string` <br/>
Default: `"{{tag}}Hooks"`


### withDevtools
Include `@hookform/devtools

Type: `boolean` <br/>
Default: `false`
