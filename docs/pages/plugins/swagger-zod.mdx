# @kubb/swagger-zod

With the Swagger Zod plugin you can use [Zod](https://zod.dev/) to validate your schema's based on a Swagger file.

## Installation

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash copy
    pnpm i -D @kubb/swagger-zod
    ```
  </Tab>
    <Tab>
    ```bash copy
    npm i -D @kubb/swagger-zod
    ```
  </Tab>
    <Tab>
    ```bash copy
    yarn add -D @kubb/swagger-zod
    ```
  </Tab>
</Tabs>

## Options


### output
Output to save the Zod schemas.

Type: `string` <br/>
Default: `"zod"`

### groupBy
Group the clients based on the provided name.
Tag will group based on the operation tag inside the Swagger file

Type: `'tag'` <br/>
