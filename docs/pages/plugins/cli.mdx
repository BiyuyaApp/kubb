
## Options

```mdx
Usage: kubb [options]

Generate

Options:
  -c, --config <path>   Path to the Kubb config
  -d, --debug          Debug mode (default: false)
  -w, --watch          Watch mode based on the input file
  -V, --version        Output the version number
  -h, --help           Return a help with all commands that are possible.
```

### --config (-c)

Path to the Kubb config.

```sh copy
npx kubb -c config.ts
```

### --debug (-d)

Debug mode (default: false).

```sh copy
npx kubb -d
```

```sh copy
npx kubb -d
```

### --watch (-w)

Watch mode based on the input file.

```sh copy
npx kubb -w
```

### --version (-v)

Return version used by Kubb.

```sh copy
npx kubb -v
```

### --help (-h)

Return a help with all commands that are possible.

```sh copy
npx kubb -h
```
