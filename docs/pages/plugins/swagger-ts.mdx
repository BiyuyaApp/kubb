# @kubb/swagger-ts

With the Swagger TypeScript plugin you can create [TypeScript](https://www.typescriptlang.org/) types based on a Swagger file.

## Installation

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash copy
    pnpm i -D @kubb/swagger-ts
    ```
  </Tab>
    <Tab>
    ```bash copy
    npm i -D @kubb/swagger-ts
    ```
  </Tab>
    <Tab>
    ```bash copy
    yarn add -D @kubb/swagger-ts
    ```
  </Tab>
</Tabs>



## Options

### output
Relative path to save the TypeScript types.
When output is a file it will save all models inside that file else it will create a file per schema item.

Type: `string` <br/>
Default: `"models"`

### groupBy
Group the clients based on the provided name.
Tag will group based on the operation tag inside the Swagger file

Type: `'tag'` <br/>

### groupBy
Group the clients based on the provided name.
Tag will group based on the operation tag inside the Swagger file

Type: `'tag'` <br/>

### enumType
Choose to use enum or as const for enum

Type: `'enum' | 'asConst'` <br/>
Default: `asConst`
